# TicTacToe
![TicTacToe](/img/TicTacToe.png?raw=true)

## Quick Start

Dependencies: [SDL2], [SDL2_ttf]
```console
$ cd TicTacToe/src/
$ chmod +x build.sh
$ ./build.sh
$ ./TicTacToe
```
